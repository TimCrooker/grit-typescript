# grit-typescript

> my awesome new Grit generator

## Usage

Install [Grit](https://github.com/TimCrooker/projenerator) first.

```bash
yarn global add grit-cli
# or
npm i -g grit-cli
```

### From npm

```bash
grit typescript my-project
```

### From git

```bash
grit TimCrooker/grit-typescript my-project
```
